Sobre el grupo

Organizado por el USFQ Data Hub, este grupo reúne a profesores de economía, ciencia política, sociología, antropología, relaciones internacionales, ciencias de la computación y matemática para explorar juntos cómo la inteligencia artificial y el machine learning están transformando las ciencias sociales — tanto como herramientas metodológicas como objetos de estudio.

Cada sesión gira alrededor de una lectura principal accesible para toda la audiencia, con una lectura opcional para quienes quieran profundizar. Las sesiones alternan entre énfasis técnico crítico/social y puente — lecturas que conectan ambas dimensiones.
